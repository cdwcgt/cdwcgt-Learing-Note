# HelloWorld

1.建立一个空文件，扩展名为```.java```

2.命名为Hello

代码:

```Java
public class Hello{
	public static void main(String[] args){
		System.out.print("Hello,World!");
	}
}
```

3.用```Javac ".java文件名"```编译

4.用```Java "文件名"```运行**不用带``.class``**



## 注意事项

1.Java**大小写敏感 **，单词**大小写**不能搞错。

2.类名和文件名**必须一样**

3.不能使用**中文**符号
